Homepage UI Documentation
=========================

The homepage is the main public entry point of Massage Nexus and should function as a content-rich directory portal rather than a simple marketing landing page. It should help users immediately search, browse, discover, compare, read, and engage with spas, therapists, articles, promos, products, polls, quizzes, reviews, and platform activity. The homepage should feel active and useful even before a user logs in, while also changing parts of the interface when a registered user, owner, therapist, editor, or administrator is logged in.

Page Layout Overview
--------------------

The homepage is divided into four major layout areas: the top header/navigation area, the main content area, the right sidebar, and the footer. The top header contains branding, navigation, and account access; the main content area contains search, featured listings, browse sections, articles, reviews, promos, products, trending items, and profile updates. The right sidebar contains platform statistics, sponsored placements, engagement widgets, quizzes, quotes, and campaign cards. The footer contains secondary navigation, business links, therapist links, social links, legal links, language selection, copyright, and the footer logo.

--------------------------------------------------------------------------------

1. Top Header and Navigation
============================

Placement: Top of the page, fixed or visually prominent across the full width of the screen.

The top header contains the Massage Nexus logo on the left, followed by the main public navigation links: Home, Directory, Articles, Campus, and Promos. The extra duplicate brand text beside the logo should be removed so the header does not repeat the application name unnecessarily. On the right side, guests see Log In and Sign Up buttons, while logged-in users see their avatar, display name, user badge or rank summary, notifications, messages, and dashboard/account menu.

--------------------------------------------------------------------------------

2. Search and Sponsored Hero Area
=================================

Placement: Immediately below the top navigation, occupying the most visible upper portion of the homepage.

The search area is the primary homepage action and should remain the first major thing users notice. Immediately above the search panel, a row of target-type tabs (Spas, Therapists, Services, Articles, Reviews, Products) lets the user choose what they are searching for before they type; selecting a tab updates the search field's placeholder text and the hidden search-type parameter submitted with the query, so results open already scoped to that type instead of requiring a second filtering step after the results page loads. The panel itself contains a query field for the selected type plus a location field labeled with wording such as “City, area, or landmark,” not zip code. This section may use a branded Ink & Ember gradient background, an optional photographic hero image (with a dark navy overlay preserving text contrast) as a brand-approved alternative treatment, or a premium sponsored hero placement behind or beside the search panel, allowing paying listings to receive high visibility without removing the search-first purpose of the homepage.

--------------------------------------------------------------------------------

3. Search Quick Filters
=======================

Placement: Inside or directly below the main search panel.

Quick filters appear as small action buttons below the search fields and help users narrow results quickly without opening advanced filters. Suggested quick filters include Open Now, Near Me, Promos, Verified, and Home Service. If a user clicks Near Me, the website may request browser location permission; if the user refuses, the system should fall back to manual city, area, or landmark search.

--------------------------------------------------------------------------------

4. Featured Spas
================

Placement: Main content area, directly below the search/hero area and before browse sections.

Featured Spas should appear high on the homepage because this section can support premium visibility, sponsored placement, or selected editorial promotion. Each spa card should show the spa image, name, location, rating, review or comment count, open/closed status, opening or closing time, signature services, and labels such as Premium or Verified. Adult-oriented or sensual-service listings may exist elsewhere in the broader directory if handled responsibly, but they should not be allowed in homepage Featured Spas or premium hero placements.

--------------------------------------------------------------------------------

5. Featured Therapists
======================

Placement: Main content area, beside or directly after Featured Spas.

Featured Therapists highlights selected or premium therapist profiles and should appear near the top of the homepage for visibility. Each therapist card should show a profile image, therapist name, rating, review or comment count, specialties, location or service area, availability hint if available, and verification label. This section gives therapists direct visibility separate from spa profiles and can later support premium therapist placement.

--------------------------------------------------------------------------------

6. Browse by Area
=================

Placement: Main content area, below Featured Spas and Featured Therapists.

Browse by Area allows users to explore listings by city, district, neighborhood, region, or service area without typing into the search box. It should appear before Browse by Massage Type because location is often one of the first practical concerns when choosing a spa or therapist. Area chips should be simple, clickable, and easy to update based on available data or user location.

--------------------------------------------------------------------------------

7. Browse by Massage Type
=========================

Placement: Main content area, below Browse by Area.

Browse by Massage Type allows users to explore service categories such as Swedish, Shiatsu, Thai, Hilot, Reflexology, Hot Stone, Deep Tissue, Aromatherapy, Ventosa, and other massage-related categories. Each massage type may use a small icon or visual marker to make the section easier to scan. Clicking a type should lead to a filtered directory page or service category page showing relevant spas, therapists, articles, and possibly products.

--------------------------------------------------------------------------------

8. Latest Articles
==================

Placement: Main content area, below the browse sections.

Latest Articles displays recent guide-style, educational, or editorial content related to massage, spa visits, wellness, aftercare, safety, service types, and platform usage. Each article item should show a thumbnail image, article title, date, category, and optionally a short one-line excerpt if space allows. The section should include a View All link leading to the Articles area.

--------------------------------------------------------------------------------

9. Latest Reviews
=================

Placement: Main content area, near Latest Articles.

Latest Reviews makes the homepage feel active by showing recent user experiences from spa or therapist reviews. Each review card should show the reviewer display name or avatar, rating, related spa or therapist, short excerpt, and time posted. The section should focus on review activity rather than giving permanent free promotion to a listing.

--------------------------------------------------------------------------------

10. Promos and Deals
====================

Placement: Main content area, below Latest Articles and Latest Reviews.

Promos and Deals displays active discounts, packages, vouchers, free upgrades, limited-time offers, or seasonal bundles. Each promo card should show an image, offer label such as “15% Off” or “Free Upgrade,” business name, short description, and validity period. If there are no active promos, this section can be hidden or replaced with a neutral discovery section so the homepage does not look empty.

--------------------------------------------------------------------------------

11. Wellness Finds
==================

Placement: Main content area, near Promos and Deals.

Wellness Finds is the homepage preview of product discovery, even if the full shop feature is not emphasized in the main navigation yet. It can display massage oils, spa tools, towels, aromatherapy items, eye pillows, aftercare products, or other wellness-related items. Each item should show a product image, name, price, rating, and possibly seller or sponsor label when applicable.

--------------------------------------------------------------------------------

12. Trending Spas
=================

Placement: Main content area, below promos/products or in a compact list row.

Trending Spas shows listings currently receiving notable attention based on views, searches, saves, reviews, follows, clicks, or other engagement signals. The wording “Trending” is preferred over “Most Popular” because it feels current and temporary rather than permanently ranking the same listings. Each item can show rank number, thumbnail, spa name, location, rating, and review count.

--------------------------------------------------------------------------------

13. Trending Therapists
=======================

Placement: Main content area, beside or near Trending Spas.

Trending Therapists shows therapist profiles currently receiving notable attention from users. Each item can show rank number, avatar, therapist name, location or service area, rating, and review count. This section should rotate based on a defined time period such as this week or this month so it does not become a static free promotion area.

--------------------------------------------------------------------------------

14. New on Massage Nexus
========================

Placement: Main content area, below trending sections.

New on Massage Nexus displays recently added spa, therapist, service, product, or article records for a limited time. This replaces the more plain label “Recently Added” and gives the section a more branded feel. Items should only remain here within a defined recency window so the homepage does not become a permanent promotional space for old additions.

--------------------------------------------------------------------------------

15. Freshly Updated Profiles
============================

Placement: Main content area, beside or near New on Massage Nexus.

Freshly Updated Profiles displays listings with meaningful recent updates, such as updated hours, new services, new photos, revised contact details, ownership claim changes, verification updates, or status changes. The system should not show tiny internal edits or insignificant corrections here. This section helps users see that the platform is actively maintained and that listing information is being refreshed.

--------------------------------------------------------------------------------

16. Wellness Journey Panel
==========================

Placement: Top of the right sidebar, aligned near the search/hero area.

The statistics panel shows platform activity numbers such as total spas, therapists, reviews, or other useful public counters. It should include a subtle visual accent, background pattern, icon, or illustration so it does not look plain. The purpose is to create confidence that the platform has growing coverage and active information.

--------------------------------------------------------------------------------

17. Sponsored Placement Blocks
==============================

Placement: Right sidebar and selected homepage areas, especially after the statistics panel and between engagement widgets.

Sponsored placement blocks provide tasteful monetization spaces without making the homepage look overloaded with advertisements. These blocks may promote premium listings, wellness products, campaigns, campus content, therapist education, or advertiser messages. Each placement should be clearly labeled as Sponsored or Advertisement Space and should not interrupt the user’s ability to search, browse, or read.

--------------------------------------------------------------------------------

18. Community Pulse
===================

Placement: Right sidebar, after an initial sponsored placement.

Community Pulse is a rotating poll or survey widget that asks users a short question and encourages quick participation. After a user answers, the system shows the result briefly, then after a short delay a new unanswered item replaces the completed one in place if the user is still on the page. Rotation is per user and completion-aware: any page showing the widget serves items the user has not yet answered, never items they already completed. This feature helps collect lightweight user opinions and makes the homepage feel interactive. Participation rules, result recording, and rotation behavior are owned by docs/08-community/survey-system.txt.

--------------------------------------------------------------------------------

19. Campus Quiz
===============

Placement: Right sidebar, after Community Pulse and a sponsored placement.

Campus Quiz shows a short quiz question connected to Wellness Campus learning. When a user submits an answer, the system shows whether the answer is correct with a short explanation, then after a short delay a new unanswered question replaces the completed one in place if the user is still on the page. Rotation is per user and completion-aware: any page showing the quiz widget serves questions the user has not yet answered, never questions they already completed. Logged-in users may earn points and have their quiz results recorded, while guests can answer casually and be invited to sign in for saved progress. Participation rules, result recording, rotation behavior, and the open Campus-integration decision are owned by docs/10-education/quiz-system.txt.

--------------------------------------------------------------------------------

20. Quote of the Day
====================

Placement: Right sidebar, usually below the quiz and sponsored placements.

Quote of the Day displays a rotating wellness, relaxation, massage, self-care, or motivational quote from an approved quote collection. The quote should come from a database rather than a plain text file so it can support categories, languages, authors, sources, active dates, seasonal use, and moderation. This section gives the homepage a calmer editorial touch between more functional sections.

The proposed database structure for this feature is the quote_main collection, documented in the PHP-readable structure guide at docs/04-architecture/structure-guide/quote_main.php, with its quote categories defined by the type_quote_category taxonomy record in data/taxonomy/massage_nexus/content_classification.json. The current implementation still renders one temporary hard-coded sample quote; replacing it with quote_main-backed rotation is pending implementation. Quote sourcing, attribution, rotation, and administration rules are owned by docs/09-content/quote-system.txt. Unlike Community Pulse and Campus Quiz, quotes have nothing for the user to complete, so quote rotation is time-based rather than completion-aware.

--------------------------------------------------------------------------------

21. Spa Day Section
===================

Placement: Main content area or right sidebar, depending on campaign timing and importance.

The Spa Day section appears when the annual Spa Day campaign is active or near its campaign window. It may promote Spa Day articles, offers, participation prompts, featured content, or campaign graphics. Outside the relevant period, the section can be hidden so the homepage does not feel outdated.

--------------------------------------------------------------------------------

22. Logged-In User Header Variant
=================================

Placement: Top-right account area, replacing Log In and Sign Up.

When a normal registered user is logged in, the header should show the user’s avatar, display name, and a compact identity label such as Member, Verified Member, Premium User, rank, or point summary. The user menu should provide shortcuts to dashboard, saved listings, messages, notifications, profile, settings, and log out. Owners, therapists, editors, moderators, and administrators should use the same login area, but their dashboard menu should change based on role.

--------------------------------------------------------------------------------

23. Footer
==========

Placement: Bottom of the homepage, full width.

The footer contains secondary navigation and should be organized into clear columns rather than becoming a random link dump. Suggested columns include Compass Guide, For Spa Owners, For Therapists, Connect With Us, and legal/platform links; the Massage Nexus logo should appear on the right side of the footer to balance the top-left logo placement. The footer should show the correct year, social media links, language selector, Terms of Use, Privacy Policy, Cookie Policy, and copyright notice.

--------------------------------------------------------------------------------

24. Footer: Compass Guide
=========================

Placement: Footer column.

Compass Guide contains user-facing help and orientation links such as How It Works, Search Tips, Wellness Glossary, and FAQ. This replaces the need to put Help or About in the main navigation. The tone should feel like a platform guide rather than a generic support page.

--------------------------------------------------------------------------------

25. Footer: For Spa Owners
==========================

Placement: Footer column.

For Spa Owners contains links for business users who want to list, claim, manage, or promote their spa. Suggested links include List Your Spa, Claim Your Profile, Promote Your Business, and Platform API. This column supports the business side of Massage Nexus without making the homepage feel like it is only for owners.

--------------------------------------------------------------------------------

26. Footer: For Therapists
==========================

Placement: Footer column.

For Therapists contains links for massage therapists and wellness practitioners who want to join, claim a profile, understand platform rules, or access learning resources. Suggested links include Join as Therapist, Claim Your Profile, Therapist Guidelines, and Wellness Campus. The Guidelines link can cover profile rules, service boundaries, media rules, conduct, verification, and professional expectations.

--------------------------------------------------------------------------------

27. Footer: Connect With Us
===========================

Placement: Footer column or footer center area.

Connect With Us contains social media icons and links for platforms such as Facebook, Instagram, TikTok, YouTube, Pinterest, or others chosen later. It can also include the language selector if the layout works better visually. Social links should support brand presence and campaigns without distracting from the main directory purpose.

--------------------------------------------------------------------------------

28. Footer: Legal and Policy Links
==================================

Placement: Bottom footer row.

The bottom footer row should include Terms of Use, Privacy Policy, Cookie Policy, and other legal links added later. The Privacy Policy explains personal data handling, while the Cookie Policy explains cookies, tracking, analytics, ads, and cookie preferences. These should be accessible but not visually dominant.

--------------------------------------------------------------------------------

29. Footer: Language Selector
=============================

Placement: Footer area, usually near the social links or bottom-right area.

The language selector allows users to choose the interface language or regional language setting. It may display a country flag or language label, but the system should avoid implying that language and country are always the same thing. This supports the multilingual direction of Massage Nexus. The listed options should follow the rollout order defined in the Language Strategy document (Foundation, then First International Expansion, then Second International Expansion) rather than an arbitrary list; a language appearing in the selector does not by itself mean its interface strings are translated, only that it is on the approved rollout path.

--------------------------------------------------------------------------------

30. Cookie Preference Banner
============================

Placement: Bottom of the screen when required, not part of the permanent homepage layout.

The cookie preference banner appears when the site uses non-essential cookies such as analytics, advertising, personalization, or tracking. It should be small, clear, and not annoying, with options such as Accept All, Reject Non-Essential, and Manage Preferences. The banner should avoid blocking the homepage unless legally necessary.

--------------------------------------------------------------------------------

Homepage Content Behavior Notes
===============================

The homepage should avoid becoming bare when optional sections have no content. Core sections such as search, browse, latest articles, and footer should always be available, while optional sections such as promos, sponsored placements, Spa Day, Wellness Finds, New on Massage Nexus, and Freshly Updated Profiles may appear only when content exists. The page should remain useful whether the platform has a small early dataset or a large mature dataset.

Homepage Monetization Notes
===========================

Homepage monetization should prioritize premium listings, sponsored placements, featured visibility, and advertiser cards rather than overwhelming the page with generic ads. Sponsored areas should be clearly labeled and visually integrated, not intrusive. Paying listings may receive visibility in the sponsored hero area, Featured Spas, Featured Therapists, side placements, product discovery, or campaign sections, depending on the paid placement rules later defined.

Homepage Search Notes
=====================

The search system should handle broad queries across spas, therapists, services, reviews, articles, and locations, but results should be organized clearly so users understand what type of result they are seeing. The homepage hero implements this through target-type tabs (Spas, Therapists, Services, Articles, Reviews, Products) selected before the query is submitted; the results page may still offer its own filters and result tabs for refining within a type or broadening back to a mixed view. Location search should use terms like city, area, or landmark rather than zip code.
