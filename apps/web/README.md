# Massage Nexus Web Application

This directory contains the Laravel web application for Massage Nexus.

Repository-wide setup, validation, documentation, contribution, and ownership information is in
the root [`README.md`](../../README.md) and [`AGENTS.md`](../../AGENTS.md).

Run Composer, Artisan, PHPUnit, Pint, npm, and Vite commands from this directory unless a command
explicitly targets the repository root.

Massage Nexus source code and project materials are private and proprietary. Laravel and other
third-party dependencies retain their own licenses; using an MIT-licensed dependency does not
license the Massage Nexus application under MIT.
